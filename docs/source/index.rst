Welcome to Metabarcoding at Cefas!
===================================

Here at Cefas, Metabracoding is a key tool in many genomics projects.
This tutorial space is designed to help anyone at Cefas who is working with Metabarcoding to get started with the bioinformatics tools and skills needed to carry out this work.

As a starting point, check out the :doc:`dada2` section for information on this widely used software, including
how to :ref:`installation` the program.

.. note::

   This project is under active development.
